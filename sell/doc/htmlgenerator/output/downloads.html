Additional Features

Rates can be imported/updated using CSV files.

How Upgrading from Free Version

  • make a backup of current Asterisell installation;
  • copy files inside current Asterisell installation;
  • enable “PhpRateImportFromCSV”, uncommenting the “available/phpRates” related line in “apps/asterisell/config/app.yml”;
  • execute “./symfony cc” and “./symfony fix-perms” inside current Asterisell installation directory;

Versioning Scheme

  asterisell-W-X.Y.Z

  • W can be something like stable/testing/development;
  • X is major sequence: it changes only if the application update requires a conversion of the database, in particular of CDR table;
  • Y is minor sequence: it changes if the new version introduces new features;
  • Z is maintenance sequence: it changes if the new version introduces bug-fixes or little improvements of already existing features;
