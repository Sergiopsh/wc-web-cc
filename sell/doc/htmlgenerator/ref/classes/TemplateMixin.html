Module TemplateMixin
In: rassmalog.rb

In order to mix-in this module, an object must:

  1. have an associated template file (whose basename is specified by template_name) in the config/ directory.

    A default template_name, which uses the name of the mixer‘s class, is provided.

  2. define a url method which returns a relative URL to it. 
  A default #url, which uses #name, is provided.
